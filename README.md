# 投研协作平台 (Research Collaboration Copilot Platform)

> 面向基金公司研究所的智能投研协作平台，基于 OpenClaw Agent 框架，集成 AI 辅助研究、自动文档解读、智能预警推送等核心能力。

## 🛠 技术栈

| 层级 | 技术选型 | 说明 |
|------|---------|------|
| Frontend | Next.js 14 + TypeScript + Tailwind CSS + shadcn/ui | 现代化 React 全栈框架 |
| Backend | Python 3.11 + FastAPI + SQLAlchemy 2.0 | 高性能异步 API 服务 |
| Database | PostgreSQL 16 + pgvector | 关系型数据 + 向量检索 |
| Cache/MQ | Redis 7 | 缓存 / 事件总线 / 任务队列 |
| AI Engine | OpenClaw + LangChain | Agent 编排 + RAG 检索增强生成 |
| File Storage | MinIO (S3 Compatible) | 文档存储 |
| Task Queue | Celery + Redis | 异步任务处理 |
| Auth | JWT + RBAC | 基于角色的访问控制 |
| Deploy | Docker + Docker Compose | 容器化部署 |

## 🚀 快速开始

### 环境要求
- Node.js >= 20
- Python >= 3.11
- Docker & Docker Compose
- PostgreSQL 16 (或使用 Docker)
- Redis 7 (或使用 Docker)

### 一键启动 (Docker)
```bash
# 克隆项目
git clone <repo-url>
cd investment-research-platform

# 配置环境变量
cp .env.example .env
# 编辑 .env 填入必要配置

# 启动所有服务
docker-compose up -d

# 初始化数据库
docker-compose exec backend python -m app.db.init_db

# 访问
# 前端: http://localhost:3000
# API: http://localhost:8000/docs
```

### 本地开发
```bash
# 后端
cd backend
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
uvicorn app.main:app --reload --port 8000

# 前端
cd frontend
npm install
npm run dev
```

## 👥 角色说明

| 角色 | 标识 | 说明 |
|------|------|------|
| 研究员 | `researcher` | 日常研究、文档阅读、结论维护 |
| 基金经理 | `pm` | 查看重点变化、发起问题、决策支持 |
| 研究所领导 | `leader` | 团队管理、效率看板 |
| 系统管理员 | `admin` | 用户权限、系统配置 |
