## 投研协作平台 MVP 项目代码骨架搭建

基于 v2.0 PRD 完成了完整项目骨架搭建，技术栈和关键决策：

### 技术栈
- **Frontend**: Next.js 14 + TypeScript + Tailwind CSS
- **Backend**: Python 3.11 + FastAPI + SQLAlchemy 2.0 (async)
- **Database**: PostgreSQL 16 + pgvector (向量检索)
- **AI Engine**: OpenClaw Agent 框架 (LLM 编排)
- **Event Bus**: Redis Pub/Sub (事件驱动架构)
- **Task Queue**: Celery + Redis (异步任务)
- **File Storage**: MinIO (S3 兼容)
- **Deploy**: Docker Compose 一键部署

### 已完成模块
1. **数据库模型** (16 张表): User, Company, ResearchTask, Document, DocumentChunk(向量), ConclusionCard, Assumption, QuestionThread, Alert, AlertDelivery, ResearchTemplate, QABookmark, ViewpointConflict, ActivityLog, CopilotConversation
2. **API 路由** (12 个 Router): auth, companies, documents, tasks, conclusions, assumptions, questions, copilot, alerts, workbench, dashboard, management
3. **AI Agent** (4 个): DocumentAnalysisAgent(文档解读), CopilotAgent(问答), PeerComparisonAgent(同行对比), AlertClassificationAgent(事件分级)
4. **事件总线**: Redis Pub/Sub, 定义了 15+ 事件类型
5. **前端页面** (4 页): 登录页, 研究员工作台, 基金经理驾驶舱, 管理层效率看板
6. **Celery 异步任务**: 文档分析、邮件推送、假设偏离定时检查
7. **种子数据**: 5 个演示用户、3 个示例公司、5 个预置研究模板
8. **Docker Compose**: 7 个服务 (postgres, redis, minio, backend, celery-worker, celery-beat, frontend)

### 项目约定
- 项目基于 OpenClaw 框架，面向基金公司研究所推广
- RBAC 4 角色: researcher / pm / leader / admin
- 所有 AI 输出带 citations + confidence_level 字段
- 所有核心实体带 created_at/updated_at/created_by/updated_by 审计字段
